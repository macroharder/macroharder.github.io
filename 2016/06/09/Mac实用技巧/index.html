发表于 2016-06-09   |  

快捷键篇

作用 快捷键
最小化所有窗口 command + option + M + H
不带格式粘贴 command + option + shift + V
窗口截图带阴影 shift + command + 4 + 空格
切换并前置最小化的窗口 command + tab 切换到对应的窗口后 command 键不要松，按下 option 键后再一起松
显示选中文件简介 command + I
输入特殊符号 control + command + space
音量轻微调节 option + shift + F10/F11
剪切 command + C –> command +option + V

命令篇

作用 命令
显示隐藏文件夹 defaults write com.apple.finder AppleShowAllFiles -bool true;killall Finder
隐藏隐藏文件夹 defaults write com.apple.finder AppleShowAllFiles -bool false;killall Finder
在终端中打开当前目录 open .
强制Safari在新建窗口中打开链接 defaults write com.apple.Safari TargetedClicksCreateTabs -bool true
修改通知停留时间 defaults write com.apple.notificationcenterui bannerTime 时间(单位为秒)
